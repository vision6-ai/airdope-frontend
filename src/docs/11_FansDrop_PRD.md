# Public Drop PRD (Fan View)

## 1. Overview
The **Public Drop Page** (`FansDropPage`) is the landing page that fans see when they click a drop link. It is optimized for conversion (RSVP).

## 2. User Stories
*   **As a Fan**, I want to see the event details (Date, Time, Location/Link).
*   **As a Fan**, I want to RSVP using my Email or Phone number.

## 3. Key Features
*   **Visuals:** Large, immersive artwork/flyer.
*   **Event Details:** Title (e.g., "Mayan Warrior - Guadalajara"), Date, Host logo.
*   **RSVP Form:**
    *   **Input Mode:** Toggle between Email and Phone.
    *   **Submission:** Collects contact info.
    *   **Legal:** Terms/Privacy links.
*   **Success State:**
    *   "Check your email/phone" confirmation.
    *   **Share Options:** "Share", "Make a drop like this".

## 4. User Flow
1.  **Arrival:** Fan clicks a shared link and lands on the page.
2.  **Interaction:** Fan views artwork and details.
3.  **Conversion:** Fan enters email and clicks RSVP.
4.  **Confirmation:** Fan sees success message.
